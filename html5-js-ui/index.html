Seeq Data Lab Functions Example

Welcome to an example of calling a Seeq Data Lab Function (DLF) from HTML5 and JavaScript. It is not fancy but hopefully covers the concepts.

The following buttons support DLF demo. The 'Seeq Login' button must be clicked first to log into Seeq and then the others to 'Do SDL Work' and 'Get Work' can be called after.

For more detail on this demo see: Go to Seeq Data Lab Functions Example

Click to log into Seeq

Using the DLF API click to tell Seeq Data Lab to do the work of retrieving a DataFrame from a search for Example signals.

Click to retrieve current DLF work output. Note: If DLF has not been run yet then work output will indicate no results yet.

Perform complete login->work->results on one action.
